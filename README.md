#  `BASES DE DATOS (EST - 230)`

## `Objetivo General`
> Diseñar e implementar una Base de Datos aplicando herramientas conceptuales (Modelos de Datos) y computacionales (Sistemas Gestores de Bases de Datos, Lenguajes de Programación y Lenguajes de Datos).


## `Contenido Temático`
> - Tema 1 : Modelo de Datos
> - Tema 2 : Modelo Entidad - Relación
> - Tema 3 : Modelo Relacional
> - Tema 4 : Algebra Relacional
> - Tema 5 : Lenguaje SQL 
> - Tema 6 : Lenguaje R y Bases de Datos
